A project I've been working on

Coding Basics

Coding Basics is a curriculum for middle schoolers to help teach kids a wide variety of programming at a young age.

Coding Basics will assist students in discovering their passion for coding, as well as help them determine courses for future career opportunities such as graphic design and video game development. Students will develop computer coding skills by learning languages such as Batch, Python, C# & XML, and HTML. Using these skills, students will create programs using reason and creativity.

My best work

PytchScript

One of my favorite projects is PytchScript.

PytchScript is a scripting language I wrote in python designed to be as powerful as Python, yet have the syntax of Batch. It is designed to help kids learn how to understand the logic of a program. It supports everything from forloops to GUI interfaces. Pytch is an ongoing project that is still being updated. You can view the documentation here.
